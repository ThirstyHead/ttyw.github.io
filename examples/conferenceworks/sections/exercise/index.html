Exercise: Sections and Web Components

In this exercise, you'll make changes to the theme. You'll also refactor the header into a reusable Web Component.

Steps

Tweak the theme

  1. The design team is asking you to change the color of "Conference" in "ConferenceWorks" from red to green.
  2. Ensure that the color change takes effect in both the header and the footer.

Refactor the header

  1. Run the tests for 02-Sections.
    Type gauge run specs/02-sections
  2. To view the report.
    Type open reports/html-report/index.html
  3. There should be one failing test.
  4. To make the test pass, extract the header into a Web Component.
  5. Once you have created the new web component, run the tests again and ensure that they all pass.
