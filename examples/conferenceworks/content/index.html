Content and Responsive Web Design

In this section, we'll build out a basic page for Speaker information.

To begin, we'll explore the <nav> element and ways to style it. We'll create landing pages for the Speakers and the Schedule.

Next, we'll both see and hear the page hierarchy using <h1> - <h6> sectioning elements.

We'll add an <img>, and discuss both alt text and the more robust <figure> + <figcaption> combination. We'll also discuss the merits of using the <picture> element instead of the <img> element.
