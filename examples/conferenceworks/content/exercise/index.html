Exercise: Content and Responsive Web Design

In this exercise, you'll add several new pages to the website. You'll add links to them in the header.

Steps

Run the tests

  1. Run the tests for 03-Content.
    Type gauge run specs/03-content.spec
  2. Note the failing tests. (You'll fix them in this exercise.)

Create new pages and link to them

  1. Create new subdirectories speakers and schedule off of the root directory.
  2. Copy index.html into each new subdirectory.
  3. Change the <title> to include "Speakers" and "Schedule". Also, add an <h1> to each page that contains the same text.
  4. Add a <nav> block to <cw-header> with links to your new pages.

Build out the Speakers landing page

  1. The raw ingredients you'll need for this exercise are in __rawdata, including images, bio.txt, and talks.txt for all six speakers.
  2. Add each speaker's image, name, and title to the Speaker landing page.
  3. Add a link from each speaker's "card" to their details page.
  4. Create a speaker details page for each speaker.
