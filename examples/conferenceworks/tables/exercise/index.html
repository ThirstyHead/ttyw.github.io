Exercise: Tables and LocalStorage

In this exercise, you'll build out the Schedule dynamically from a microservice.

Steps

Build out the table

  1. Using the skills you acquired, build out a static "shell" of the schedule (i.e. just the rooms and slots, not the talks or speakers).

Mock out the JSON data

  1. Create the mock JSON data in the webpage, and write it out to localStorage.

Fetch the JSON data from the microservice

  1. Replace the mocked JSON data with fetched data, using async/await.
