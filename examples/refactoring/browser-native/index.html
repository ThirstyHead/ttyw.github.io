Browser-Native Tech

In this section, we'll discuss the merits of using standards-based, browser-native technologies -- HTML, CSS, and JavaScript -- over ephemeral third-party web frameworks. We'll create highly-cohesive, loosely-coupled, well-encapsulated web components. And we'll write User Journey Tests with Gauge and Taiko to ensure that our web-based refactorings of an existing website are done safely, without introducing new bugs or regressions to a fully-functional (but less than optimized) website called GroceryWorks.
