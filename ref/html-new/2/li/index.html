5.6.1. Unordered List: UL, LI

[since: HTML 2.x]

[source: RFC 1866, published November 1995, https://www.ietf.org/rfc/rfc1866.txt]

The <UL> represents a list of items -- typically rendered as a bulleted list.

The content of a <UL> element is a sequence of <LI> elements. For example: 


    <UL>
    <LI>First list item
    <LI>Second list item
     <p>second paragraph of second item
    <LI>Third list item
    </UL>
